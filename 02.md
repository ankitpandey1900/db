# 🔷 Q2 (A)

---

## 1) Total workout duration per user (per day)

```sql
SELECT user_id, date, SUM(duration_in_minutes) AS total_duration
FROM workouts
GROUP BY user_id, date;
```

👉 Pattern: **GROUP BY multiple columns**

---

## 2) Procedure: workouts in date range

```sql
DELIMITER //

CREATE PROCEDURE get_user_workouts(
    IN uid INT,
    IN start_d DATE,
    IN end_d DATE
)
BEGIN
    SELECT *
    FROM workouts
    WHERE user_id = uid
    AND date BETWEEN start_d AND end_d;
END //

DELIMITER ;
```

---

## 3) Users + total workouts

```sql
SELECT u.name, COUNT(w.workout_id) AS total_workouts
FROM users u
LEFT JOIN workouts w ON u.user_id = w.user_id
GROUP BY u.name;
```

👉 LEFT JOIN → include users with 0 workouts

---

## 4) View: workout summary

```sql
CREATE VIEW workout_summary AS
SELECT u.name, w.workout_type, SUM(w.duration_in_minutes) AS total_duration
FROM users u
JOIN workouts w ON u.user_id = w.user_id
GROUP BY u.name, w.workout_type;
```

---

## 5) Users with weight loss goal

```sql
SELECT DISTINCT u.name
FROM users u
JOIN users_goal ug ON u.user_id = ug.user_id
WHERE ug.goal_type = 'weight loss';
```

---

## 6) Trigger: end_time < start_time block

```sql
DELIMITER //

CREATE TRIGGER check_time
BEFORE INSERT ON workouts
FOR EACH ROW
BEGIN
    IF NEW.end_time < NEW.start_time THEN
        SIGNAL SQLSTATE '45000'
        SET MESSAGE_TEXT = 'Invalid time';
    END IF;
END //

DELIMITER ;
```

```sql
DELIMITER //

CREATE FUNCTION total_steps(uid INT, d1 DATE, d2 DATE)
RETURNS INT
DETERMINISTIC
BEGIN
    DECLARE total INT;
    SELECT SUM(step_count)
    INTO total
    FROM steps
    WHERE user_id = uid
    AND tracking_date BETWEEN d1 AND d2;
    RETURN total;
END //

DELIMITER ;
```

## 5) Cursor

```sql
DELIMITER //

CREATE PROCEDURE list_users()
BEGIN
    DECLARE done INT DEFAULT FALSE;
    DECLARE uname VARCHAR(50);

    DECLARE cur CURSOR FOR SELECT name FROM users;
    DECLARE CONTINUE HANDLER FOR NOT FOUND SET done = TRUE;

    OPEN cur;

    loop1: LOOP
        FETCH cur INTO uname;
        IF done THEN LEAVE loop1; END IF;
        SELECT uname;
    END LOOP;

    CLOSE cur;
END //

DELIMITER ;
```

## 6) Procedure: insert workout

```sql
DELIMITER //

CREATE PROCEDURE add_workout(
    IN uid INT,
```

# ⚠️ What Actually Matters (Don’t Miss This)

## 🔥 Core Patterns (Repeat Everywhere)

| Pattern            | Use                  |
| ------------------ | -------------------- |
| GROUP BY + HAVING  | filtering aggregates |
| JOIN               | combining tables     |
| ORDER BY + LIMIT   | top queries          |
| PROCEDURE/FUNCTION | reuse                |
| TRIGGER            | validation           |
